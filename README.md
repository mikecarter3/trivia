# trivia

A fun trivia game.

### Dependencies (as of 11-28-2017) ###
python (2.7 or 3.6)
pillow (4.3.0)
pytesseract (0.1.7)
pyscreenshot (0.4.2)
opencv (3.3.1)
nltk (3.2.5)

### Additional system requirements ###
A screen mirroring tool for mobile device (iTools recommended)
A web browser that loads decently quickly
